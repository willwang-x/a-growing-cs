## Why

* 把操作系统知识再巩固一遍
* 这课程有趣有料
* 用户多，笔记多，便于从多角度理解内容

## How

* 目前没有时间，就有时间看一章，然后配合Anki学习


## What

他将以Linux 4.x的内核源代码为基础，用一个完整的**创业**故事，配合**生动详实**的图片，深入剖析**Linux操作系统**的方方面面。他会教你快速上手**Linux命令**，掌握学习Linux的**方法和路径**，带你详细推演Linux内核的**工作原理**，让你对Linux操作系统的**细节和全貌**都有个完整的认识，并能站在一定高度去**理解和优化**当下的工作。
